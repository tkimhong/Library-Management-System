# Library Book Borrowing System

A Java-based library management system for managing book checkouts, returns, and user accounts.

## Team Members
- **Mengkheang** - User Management Module
- **Morngdyza** - Book Management Module
- **Kimhor** - Borrowing System Module
- **Tiangkimhong** - Rules & Integration Lead

## Project Structure
```
src/
├── models/          # Data models (User, Book, BorrowRecord)
├── services/        # Business logic
├── ui/              # User interface
└── Main.java        # Application entry point
```

## Setup Instructions
1. Clone the repository
2. Open in your Java IDE (IntelliJ, Eclipse, VS Code)
3. Run `Main.java` to start the application

## Timeline
- **2025-12-05**: Documentation
- **2025-12-19**: Final project submission

## Features
- User Management (add, view, edit, deactivate)
- Book Management (add, view, update, remove, search)
- Borrowing System (checkout, return, track)
- Rules Engine (3-book limit, 14-day periods, overdue tracking)
